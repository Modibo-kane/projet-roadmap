# Projet Roadmap — De l'idée au produit

Une application web interactive et moderne conçue pour accompagner les développeurs et entrepreneurs dans la gestion de leurs projets informatiques, de la phase d'idéation jusqu'au lancement et à la croissance.

## Fonctionnalités

- **6 Phases Détaillées** : Un parcours complet allant de l'Idéation au Post-lancement.
- **Suivi de Progression** : Micro-tâches interactives avec mise à jour en temps réel des statistiques de complétion.
- **Interface Moderne (Dark Mode)** : Design élégant basé sur des nuances sombres, des dégradés "IA" (jaune et vert) et une typographie soignée.
- **Filtrage & Recherche** : Système de filtrage par phase et moteur de recherche instantané pour retrouver facilement une étape ou une tâche.
- **Export Markdown** : Fonctionnalité intégrée pour copier la roadmap au format Markdown, facilitant l'intégration dans une documentation ou un dépôt GitHub.
- **Design Responsif** : Interface optimisée pour une consultation fluide sur desktop et mobile.

## 🛠️ Technologies Utilisées

- **HTML5** : Structure sémantique pour une meilleure accessibilité.
- **CSS3** : Utilisation intensive des variables CSS pour le thème, Flexbox pour la mise en page et dégradés dynamiques.
- **JavaScript (Vanilla)** : Logique de filtrage, gestion dynamique des classes et manipulation du DOM sans dépendances externes.
- **FontAwesome 6** : Bibliothèque d'icônes vectorielles pour une identité visuelle forte.

## 🚀 Installation et Utilisation

Le projet est une application "client-side" pure, ce qui signifie qu'aucune configuration serveur n'est nécessaire.

1. Clonez ou téléchargez ce dossier.
2. Ouvrez le fichier `index.html` dans n'importe quel navigateur moderne.

## 📁 Structure des Fichiers

- `index.html` : Contenu principal et structure de la roadmap.
- `style.css` : Système de design, variables de couleurs et animations.
- `script.js` : Logique applicative (recherche, filtres, export).

## 💡 Personnalisation

Toutes les couleurs principales sont centralisées dans le bloc `:root` du fichier `style.css`, permettant de changer l'identité visuelle du projet en quelques secondes.
